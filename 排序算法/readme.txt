交换类排序：冒泡、快排
选择类排序：选择、堆排
插入类排序：插入、希尔

冒泡：遍历n-1遍，每遍比较相邻两个数，如果前面的数大于后面的，则交换，每遍的结果是最大的沉到最底

选择：n-1遍，每遍找到最小值，并放在i的位置（i是遍历的第i遍）与冒泡不同的时候，每遍主要是比较，交换只发生一次，而冒泡交换会发生多次

插入：n-1遍，第i遍的时候前面有i个已经排好，这一遍的目的就是把第i+1个数插入前面i个的正确位置处，该方法就是移动比较频繁

快排：一趟中，把数据分为两部分，左边的都比基准小，右边的都大，然后对左右两部分继续使用快排

      一趟中，令第一个为基准，设i=左边界，j=右边界，从i向右找到第一个比基准大的数，从j向左找到第一个比基准小的数，若此时i<j,则交换     
      二者，然后继续直至i>=j，此时基准左边的数都小，右边的都大，然后对左右两边进行快排
